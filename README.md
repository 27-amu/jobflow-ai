# JobFlow AI

JobFlow AI is a Python-based job application tracker and follow-up assistant designed to help users organize applications, manage recruiter communication, and track progress in one place.

## Current Features

- Job application tracking
- Application status overview
- Simple dashboard preview
- Follow-up workflow foundation

## Tech Stack

- Python
- Streamlit
- FastAPI
- SQLite
- SQLAlchemy
- Git & GitHub

## Project Goal

Build a practical product that can help job seekers manage their application pipeline and communication workflow more effectively.

## Run the Streamlit App

```bash
streamlit run app/frontend/streamlit_app.py
```
